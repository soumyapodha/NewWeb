Introduction

  • HTML is the standard markup language.
  • HTML stands for HYPER TEXT MARKUP LANGUAGE.
  • HTML is used for creating web pages.
  • HTML describe the structure of a web site.
  • It is not a programming language, it's a markup language.
  • The markup language is the collection of tags or elements.
  • HTML elements tell the browser how to display the content

HTML History

  • 1989: Tim Berners-Lee invents the Web with HTML as its publishing language
  • 1990: The time was ripe for Tim's invention
  • 1991: Open discussion about HTML across the Internet begins(september)
  • 1992: NCSA is intrigued by the idea of the Web
  • 1992: Marc Andreessen makes a brief appearance on WWW- talk(December)
  • 1993: Lou Montulli releases the Lynx browser version 2.0a(March)
  • 1993: Dave Raggett begins to write his own browser
  • 1993: The Mosaic browser is released(April)
  • 1993: Large companies underestimate the importance of the Web
  • 1994: NCSA assigns commercial rights for Mosaic browser to Spyglass, Inc.(May)
  • 1994: The first World Wide Web conference is held in Geneva, with HTML+ on show(May)
  • 1994: The Internet Engineering Task Force (IETF) sets up an HTML working group(September)
  • 1994: HTML specification for HTML 2 is released(July)
  • 1994: Netscape is formed(November)
  • 1994: The World Wide Web Consortium forms
  • 1995: HTML is extended with many new tags
  • 1995: HTML 3 is published as an Internet Draft(March)
  • 1995: A furor over the HTML Tables specification(March)
  • 1995: Microsoft's Internet Explorer browser comes out(August)
  • 1995: Netscape submits a proposal for frames(September)
  • 1995: The HTML working group runs into problems(November)
  • 1995: Vendors unite to form a new group dedicated to developing an HTML standard(November)
  • 1995: Style sheets for HTML documents begin to take shape(November)
  • 1995: Internationalization of HTML Internet Draft(November)
  • 1995: The HTML working group is dismantled(December)
  • 1996: The HTML ERB is formed(February)
  • 1996: The W3 Consortium working draft on Scripting comes out(April)
  • 1996: Microsoft seems more interested than first imagined in open standards(July)
  • 1996: Work on `Cougar' is begun(December)
  • 1997: HTML 3.2 is ready(January)
