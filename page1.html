What is HTML element?


* A HTML element is defined by a start tag, and end with a end tag.

* Some contents are given between the start tag and end tag.

* HTML elements tell the browser how to display the content

HTML Elements:~


<element name>Content here... </elementname>

Details:~


In the above the red and yellow background is called HTML elements, where the red background is the starting tag or element and the yellow background is called ending tag or element and the green background is the plece where you can write the contents of your html documents.


Some examples of HTML elements:~


<h4>This is a heading element</h4>

<p>This is a paragraph element</p>

Types of HTML Elements or Tags:~


There are two types of HTMl elements;

  1. Paired tags
  2. Unpaired tags

Paired Tags

The pair tag always have starting tag and ending tag.

For examples:-

  • <html>.......</html>
  • <body>.......</body>
  • <p>.......</p>

Unaired Tags

The tags which don't have an ending tag or closing tag, this types of tags are called unpaired tags.

For examples:- <br>, <hr> etc.

Nested HTML element:~

When one HTML element is put inside another HTML element, it is called nested HTML elements.

For examples:-
<p>This is a paragraph.<i>This is a nested element</i></p>

Details:~

In the <p> tag is the root element and the <i> is the nested element. Because the <i> tag placed inside another tag, so the <i> tag is called the nested element.

Notes:~

When there is no content in between HTML tags it is called Empty HTML element. E.g. <br> is an empty element, where it is opened and closed without any content in between the tags. Please remember there is a space after br character followed by forward slash (/).
Example:- <br>, <hr>, <link> etc. are the empty element.

Basic structure of a HTML document:~
<!DOCTYPE html>
<html>
<head>
<title>Heading Example</title>
</head>
<body>

Content goes here...

</body>
</html>
