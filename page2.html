HTML use the attribute to provide additional information about the element.

  • All HTML elements have an attribute.
  • Attribute is used to provide the additional information about the element.
  • The attribute is always placed in the starter tag.

All details about attribute:~

Tag Attribute functionality Example
<a> href <a> tag defines a hyperlink and the href attribute specifies the url of the page. <a href="https://www.instagram.com/webfamily.in/?hl=en">Instagram page</a>
<img> src <img> tag is used to show to show an image in a HTML page. The src attribute specifies the path of the image that display in the page. <img src="img/logo1.jpg" >
- alt The alt attribute provide the alternative text for an image or other tag where the link will be give. If some reason the image is not display then the text will be displayed in the place of that image. <img src ="img_girl.jpg" alt ="Girl with a jacket" >
- style The style attribute is used to add styles to an element, such as color, font, size etc. <h1 style="color:red;">This is a red color heading.</h1>

There are a lots of attributes are used in HTML element, that are discussed in later.


  • Any HTML element start with a headiing.
  • It is specified as <head> tag.
  • There are 6 differnts types of heading elements are available.
  • These headings are differnt in sizes.
  • They are <h1>, <h2>, <h3>, <h4>, <h5>, <h6>
  • The <h1> is larger in size and the <h6> is smaller in size.
  • Lower the heading number is larger the heading.
  • Example:-
<!DOCTYPE html>
<html>
<head>
<title>Heading Example</title>
</head>
<body>
<h1>This is Heading1</h1>
<h2>This is Heading2</h2>
<h3>This is Heading3</h3>
<h4>This is Heading4</h4>
<h5>This is Heading5</h5>
<h6>This is Heading6</h6>
</body>
</html>
