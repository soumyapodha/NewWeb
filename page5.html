In HTML documents various types of fonts are available.

  • We are style a letter by using a tag or by declaring a font style property on the corosponding element which would be style.
  • We can bold a letter by using the <b> tag.
  • We can change a letter to italic by using the <i> tag.
  • This are all pair tag, i.e. it has starting and ending tag.
  • Example of Bold Tag:-
<!DOCTYPE html>
<html>
<head>
<title>Bold Text Example</title>
</head>
<body>
<p><b>This sentence is written in bold.<b></p>
</body>
</html>
Output:-
  • Example of Italic Tag:-
<!DOCTYPE html>
<html>
<head>
<title>Italic Text Example</title>
</head>
<body>
<p><i>This sentence is written in italic letter.<i></p>
</body>
</html>
Output:-


  • The style attribute is placed in the starting tag or element.
  • The style attribute is used to add styles to an element of an HTML document, such as color, font, size etc.
  • Syntax to write style attribute;
    <tagname style="property:value;" >
  • We can also change the background color of the element.
  • It is a CSS property.
  • Example:-
<!DOCTYPE html>
<html>
<head>
<title>Fonts Example</title>
</head>
<body>
<h3 style="color: red;">This is heading1</h3>
<h3 style="color: blue;">This is heading2</h3>
</body>
</html>
Output:-
