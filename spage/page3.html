A paragraph always start with a new line.

  • To write a paragraph use the <p> tag.
  • The <p> tag definea a paragraph
  • This is pair tag, i.e. it has starting and ending tag.
  • Example:-
<!DOCTYPE html>
<html>
<head>
<title>Paragraph Example</title>
</head>
<body>
<p>This is a paragraph</p>
<p>

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Suscipit assumenda molestiae ad veritatis maiores explicabo hic cumque, dolore nam reiciendis delectus voluptate quia nisi odio similique temporibus est consequuntur. Aspernatur maiores quidem hic. Iure!

</p>
</body>
</html>
Output:-


  • The <br> tag is used to break a line
  • This is a singular tag i.e. it has not ending tag.
  • After use <br> tag a new line will be start.
<!DOCTYPE html>
<html>
<head>
<title>Line Break Example</title>
</head>
<body>
<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Velit voluptas eius maiores quis magni ullam vero deleniti nobis at repellat molestiae exercitationem repellendus distinctio excepturi perspiciatis, natus ex consequatur cum minima quas pariatur cupiditate!</p>
